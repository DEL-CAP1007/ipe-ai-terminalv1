# Samsung T7

Starter Python project stored on the Samsung T7 drive.

Quickstart

1. Open a terminal and activate the virtualenv:

```zsh
cd /Volumes/T7/ipe-ai-terminal
source venv/bin/activate
```

2. Install dependencies (already installed in this venv):

```zsh
pip install -r requirements.txt
```

3. Run the simple CLI:

```zsh
python src/main.py --name "Your Name"
```
